# Tomodomo Block Starter

A custom gulp and webpack-based starter for building WordPress blocks. Intended to house multiple blocks in one plugin package.

## Usage

1. `nvm install && nvm use`
2. `composer install`
3. `npm i`
4. `npm run dev` or `npm run build`
